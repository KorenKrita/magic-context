## What it does

<CardGrid stagger>
	<Card title="Unbounded sessions" icon="seti:clock">
		A background Historian condenses old conversation into a compact, tiered
		history. Your session keeps going — no context wall, no lossy compaction
		pauses.
	</Card>
	<Card title="Project memory" icon="seti:db">
		Durable facts, rules, and constraints persist across sessions. Every new
		session starts already knowing your project.
	</Card>
	<Card title="Full recall" icon="magnifier">
		Nothing is lost. The agent can search and re-expand any part of the
		original conversation, project memories, and indexed git history.
	</Card>
	<Card title="Prompt-cache aware" icon="seti:pipeline">
		Every mutation is engineered around provider prompt caching — context
		management that pays for itself instead of re-billing your prefix.
	</Card>
</CardGrid>

## Works with

- **[OpenCode](https://opencode.ai)** — CLI, TUI, and Desktop
- **[Pi](https://github.com/earendil-works/pi-mono)** — terminal coding agent

One shared store: sessions, memories, and history are shared across both
harnesses on the same machine.

```bash
npx @cortexkit/magic-context@latest setup
```
